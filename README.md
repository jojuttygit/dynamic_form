
## Generate Dynamic Forms with Laravel 8, jquery and Bootstrap 4.

## Requirements

npm 8.0.0
php 7.4

## Package

laravel/ui
sweetalert2

## Operations

- User login
- Create dynamic forms
- List created forms
- Edit and update a form
- View form

Please run below commands once the repo is cloned,

- composer install
- npm install
- npm run dev
- php artisan migrate
- php artisan db:seed
- php artisan key:generate
